The WorldWideWeb is an evolving project. This log tracks necessary improvements and known issues within the HTTP daemon (server) program to ensure the stability of the multiverse. See also: Planned Features.

Required Enhancements

  • Trace Options: Integration of comprehensive trace options for better network debugging.
  • Search Integration: Ability to call shell scripts to perform deep searches on directory trees or documents.
  • Directory Browsing: Native support for browsing server directories as hypertext nodes.
  • User Context: The HTRetrieve() routine ought to be able to pick up the user node and userid to provide personalized or restricted access.

Legacy Tracking

These requirements represent the early challenges of distributed information retrieval. Resolving these issues is critical for the transition from a read-only web to a collaborative hypermedia environment.
