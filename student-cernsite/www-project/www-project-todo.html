There is much to be accomplished within the web multiverse. If you have a moment, take your pick! These tasks represent opportunities for organizations and individuals to contribute to the growth of the global web.

Project-Specific Tasklists

In addition to the general roadmap, specific technical challenges exist for our primary sub-projects:

  • Line Mode Browser
  • NeXT Browser-Editor
  • Common Library
  • Server Daemon

The Client Side

Cross-Platform Editors
While browsers exist for many platforms, sophisticated WYSIWYG editors are currently limited to the NeXT. Developing editors for other systems is vital for decentralized content creation.
Search Engines & Knowbots
Now that the web of data and indexes is established, there is a need for intelligent algorithms to perform recursive index and link tracing across the network.
Hypertext-to-Print
We require a streamlined method for generating physical books or documents directly from the web's hypertext structure.

The Server Side

Integration of Client and Server
A client capable of generating HTML becomes a general-purpose gateway, particularly useful for sites with limited external internet access.
WAIS Integration
Extending the WAIS protocol to allow for native hypertext support and conforming Universal Document Identifiers (UDIs).

Documentation & Outreach

Interactive Tutorials
Creating hypertext paths that lead prospective users through the W3 environment as a learning experience.
Canned Demonstrations
A timed, possibly narrated, walkthrough of representative locations within the web multiverse.
